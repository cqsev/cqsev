Politique de confidentialité

Un site qui parle de gouvernance des données se doit d'être transparent sur la façon dont il gère les vôtres. Voici ce que nous collectons, pourquoi, et comment nous le protégeons.

Dernière mise à jour : 20 mars 2026

Qui est responsable de ce site ?

Responsable : Oleg Chitic

Site web : cqsev.github.io

Localisation : Montréal, Québec, Canada

Contact : via le formulaire de contact ou LinkedIn

Ce site est un site personnel et professionnel qui présente le framework CQSEV pour la gouvernance des données. Il est hébergé sur GitHub Pages.

Quelles données collectons-nous ?

Nous croyons au principe de minimisation des données : ne collecter que ce qui est nécessaire. Voici exactement ce que nous collectons :

Donnée Comment Pourquoi Conservée combien de temps
Nom et courriel Formulaire de contact (Formspree) Pour répondre à votre message Jusqu'à la fin de l'échange, puis supprimé
Données de navigation Google Tag Manager / Google Analytics Comprendre quelles pages sont consultées et améliorer le site 26 mois (paramètre par défaut Google Analytics)
Adresse IP Google Analytics (anonymisée) Statistiques géographiques agrégées Anonymisée automatiquement

Ce que nous ne collectons PAS

  • Pas de création de compte ni de mot de passe
  • Pas de données de paiement
  • Pas de numéro d'assurance sociale ni de données sensibles
  • Pas de profilage publicitaire
  • Pas de partage de vos données avec des tiers à des fins commerciales

Le formulaire de contact

Quand vous utilisez le formulaire de contact, vos informations (nom, courriel, message) sont transmises via Formspree, un service tiers.

Ce que fait Formspree :

  • Transmet votre message à notre boîte courriel
  • Ne stocke pas vos données de façon permanente
  • Ne partage pas vos données avec des tiers

📄 Politique de confidentialité de Formspree

Les cookies et outils d'analyse

Un cookie est un petit fichier texte déposé sur votre appareil quand vous visitez un site web. Voici les cookies utilisés sur ce site :

Service Type de cookie Finalité Durée
Google Tag Manager Technique + Analytique Gestion des scripts d'analyse Session
Google Analytics 4 Analytique Comprendre l'utilisation du site (pages vues, durée, provenance) Jusqu'à 2 ans

Comment désactiver les cookies ?

Vous pouvez à tout moment :

  • Configurer votre navigateur pour refuser les cookies (consultez l'aide de votre navigateur)
  • Installer un bloqueur comme Google Analytics Opt-out
  • Utiliser la navigation privée pour visiter le site sans cookies persistants

La désactivation des cookies n'affecte pas le fonctionnement du site.

Assistant IA

Ce site utilise un assistant conversationnel alimenté par l'intelligence artificielle (fourni par Chatbase) pour répondre aux questions des visiteurs sur le framework CQSEV et la gouvernance des données.

Important

Les réponses de l'assistant sont générées automatiquement par une intelligence artificielle. Elles sont basées sur le contenu publié sur ce site mais peuvent contenir des inexactitudes. Pour un échange personnalisé et fiable, utilisez le formulaire de contact.

Données collectées par l'assistant

  • Le contenu de vos questions (texte que vous tapez dans le chat)
  • Informations techniques de base (navigateur, langue)

Données NON collectées

  • Aucune donnée personnelle n'est collectée sauf si vous la partagez volontairement dans le chat
  • Aucun nom, courriel ou numéro de téléphone n'est demandé par l'assistant

Utilisation des conversations

  • Les conversations peuvent être analysées de manière anonyme pour améliorer la qualité des réponses
  • Les conversations ne sont pas utilisées à des fins commerciales ou publicitaires
  • Les conversations ne sont pas partagées avec des tiers (sauf Chatbase en tant que fournisseur technique)

Pour en savoir plus : Chatbase — Politique de confidentialité.

Les liens externes

Ce site contient des liens vers des services tiers, notamment :

  • LinkedIn — pour consulter le profil de l'auteur
  • Google Fonts — pour le chargement des polices de caractères

Quand vous cliquez sur un lien externe, vous quittez notre site. Nous n'avons aucun contrôle sur la façon dont ces services gèrent vos données. Nous vous encourageons à consulter leurs politiques de confidentialité respectives.

Hébergement

Ce site est hébergé par GitHub Pages (GitHub, Inc.), qui peut collecter des données techniques (adresse IP, type de navigateur) dans ses journaux de serveur.

Vos droits

Conformément à la Loi 25 (Loi modernisant des dispositions législatives en matière de protection des renseignements personnels, Québec) et au RGPD (Règlement Général sur la Protection des Données, Europe), vous avez le droit de :

  • Accéder à vos données — Vous pouvez demander quelles données nous avons sur vous
  • Rectifier vos données — Si une information est inexacte, nous la corrigerons
  • Supprimer vos données — Vous pouvez demander la suppression de vos données personnelles
  • Retirer votre consentement — À tout moment, sans justification
  • Porter plainte — Auprès de la Commission d'accès à l'information du Québec (CAI) ou de l'autorité de protection des données de votre pays

Pour exercer ces droits, contactez-nous via le formulaire ou par LinkedIn. Nous répondons sous 30 jours.

Comment protégeons-nous vos données ?

  • HTTPS — Toutes les communications avec le site sont chiffrées
  • Minimisation — Nous ne collectons que ce qui est strictement nécessaire
  • Pas de base de données — Ce site est statique, il ne stocke aucune donnée utilisateur sur le serveur
  • Services tiers certifiés — Formspree, Google et GitHub ont des pratiques de sécurité documentées

Mineurs

Ce site n'est pas destiné aux personnes de moins de 16 ans et ne collecte pas sciemment de données personnelles de mineurs.

Modifications de cette politique

Cette politique peut être mise à jour occasionnellement. La date de dernière modification est indiquée en haut de cette page. Les modifications significatives seront signalées sur le site.

Une question ?

Si vous avez une question sur cette politique de confidentialité ou sur la façon dont nous traitons vos données, n'hésitez pas à nous contacter :

  • Via le formulaire de contact
  • Via LinkedIn

En tant que créateur d'un framework de gouvernance des données, je considère la protection de vos données personnelles comme un engagement fondamental, pas comme une formalité.
